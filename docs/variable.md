# Variable
The keywords val and mut are used to define variables.
## Immutable
The val keyword will define a variable that cannot be changed.
```koltic
val myName: String = "John";
```
## Mutable
The mut keyword will define a variable that can be changed.
```koltic
// Assignment
mut myName: String = "John";

// Reassignment
mut myName: String = "Doe";
```
