# Koltic

![Build Status](https://github.com/soteenstudio/koltic/actions/workflows/ci.yml/badge.svg)

[Koltic](www.example.com) is a programming language designed to be rigorous during compilation. It has a built-in compilation tool called the Koltic Compiler. Koltic itself is compiled into bytecode targeted for the [LightVM](https://github.com/soteenstudio/lightvm) virtual machine.
## Installation
For packages in NPM
```bash
# Install globally
npm install -g koltic

# Install locally
npm install -D koltic
```
For packages in Yarn
```bash
# Install globally
yarn global add koltic

# Install locally
yarn add koltic
```
## Get Started
Running a code
```bash
koltic --r main.lt
```
Compile a code
```bash
koltic -c main.lt
```
